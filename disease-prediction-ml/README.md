# Disease Prediction Using Machine Learning

## Overview
This project predicts diseases using machine learning based on symptoms provided in the dataset.

## Features
- Data preprocessing
- Model training
- Disease prediction
- Accuracy evaluation
- Confusion matrix generation

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Algorithm Used
- Random Forest Classifier

## Output
The model predicts diseases from symptom data and evaluates performance using accuracy and confusion matrix.
